# 2. Python 설치하기

1. Python 공식 웹사이트에 접속합니다: https://www.python.org/downloads/

2. **Download Python** 버튼을 클릭하여 최신 버전을 다운로드합니다 (Python 3.11 이상 권장)

3. 다운로드한 설치 파일을 실행합니다

4. **중요:** 설치 시 반드시 **"Add Python to PATH"** 옵션을 체크합니다

5. **Install Now**를 클릭하여 설치를 진행합니다

6. 설치 완료 후 터미널(PowerShell 또는 CMD)을 열고 다음 명령어로 확인합니다:
   ```powershell
   python --version
   ```

7. 버전 정보가 출력되면 설치 완료입니다.

# 3. GitHub Copilot Extension 설치하기

![alt text](images/image-1.png)

1. Visual Studio Code를 실행합니다.

2. 좌측 사이드바의 **Extensions** 아이콘(네모 모양)을 클릭합니다.

3. 검색 창에 "GitHub Copilot"을 입력합니다.

4. 공식 GitHub에서 제공하는 **"GitHub Copilot"** 확장을 찾아 **Install** 버튼을 클릭합니다.

5. 설치 완료 후 GitHub 계정으로 로그인합니다.
